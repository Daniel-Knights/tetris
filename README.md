## Installation

Go to the [releases page](https://github.com/Daniel-Knights/tetris/releases) and download the appropriate installer.

- MacOS: `.dmg`, `.app`
- Windows: `.msi`, `-setup.exe`
- Linux: `.deb`, `.AppImage`
